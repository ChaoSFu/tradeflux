# TradeFlux · 短线晴雨表

> **把交易者从人性的奴隶,训练成规律的操作者。**

市场不奖励最聪明的人,奖励最能管住自己的人。TradeFlux 让**客观的市场**照进**主观的你**,用 AI 居中做不带情绪的教练,把「知」一次次拉齐「行」——并在你终于知行合一后,让自己变得不再必要。

> ⚠️ 本平台提供市场数据分析与个人行为复盘,**不构成任何投资建议或买卖指令**,不做自动交易,不保证任何收益。交易决策请自行承担风险。

## 为什么能盈利 · 交易的理论

市场扣除成本后是**负和游戏**,平均的参与者必然亏钱。散户拼不过机构的信息与分析——唯一可长期依赖、且可训练的优势是**纪律:在别人做不到时,做到你本就知道该做的事**。

```
盈利 = 胜率 × 平均盈利 − 败率 × 平均亏损 − 成本
```

四条结构性规则:

1. **顺势** — 不与趋势为敌。大势为守时,逆势买入即降低胜率。
2. **截断亏损 · 让利润奔跑** — 盈亏比是命门。快砍亏损、拿住盈利,才不被处置效应反噬。
3. **固定风险仓位** — 每笔只赌固定比例。越上头下得越重,是爆仓的唯一原因。
4. **低频 · 高确定性** — 只做高确定性机会。越频繁,成本与情绪犯错越多。

> **大概率盈利 = 顺应规律 × 执行纪律**——四条全是「行为」不是「预测」。你不需要预测未来,只需要不偏离你已经知道的对的事。

## 两个引擎

- **🔭 读市场 · 客观规律引擎** — 指数均线趋势、板块轮动退潮、涨跌停与一字板、情绪温度与建议仓位。回答:现在什么环境、主线在哪、该攻还是该守。
- **🪞 照自己 · 人性纠偏引擎** — 记录你的每一笔操作,发现逆势加仓、着急回本、不舍空仓等弱点,做出客观评价。回答:我又在重复哪个错、这次怎么管住自己。

## 系统能力

| 能力 | 对应理论 | 内容 |
|------|----------|------|
| **读市场** | → 帮你顺势 · 判攻守 | 大盘趋势 · 涨跌停概览 · 强势股 · 趋势/情绪板块 · 弱转强信号 · 重点监控 |

| **AI 教练** | → 帮你缩小知行差 · 进化 | 叙事归因 · 事前良知 · 环境高危预警 · 按成熟度因材施教 |

**🧭 北极星 · 知行差指数** — 一个头条数字 = 你的操作偏离「客观市场环境」与「你自己定的规则」有多远。看着它逐月缩小,就是盈利能力在真实成长——两个引擎,最终缝成这一件事。

## 边界 · 也是判断标尺

不预测行情、不给买卖点、不做自动交易。**凡是让交易者更依赖系统、而非更成长的功能,就不做**——TradeFlux 的成功标准,是把一个情绪化的赌徒养成一个有纪律的操作者,然后功成身退。

## 技术栈 & 快速开始

后端 FastAPI + SQLAlchemy + PostgreSQL;前端 React + TypeScript + Vite + Tailwind;数据源为东方财富等公开行情接口,每日盘后/盘前自动更新。

```bash
# 后端（需 Python 3.13 + PostgreSQL）
cd backend && python3.13 -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt && cp .env.example .env
uvicorn app.main:app --reload --port 8000      # 首次启动自动建表

# 前端
cd frontend && npm install && npm run dev       # 访问 http://localhost:5173
```

接口文档:`http://localhost:8000/docs`。线上部署:`git push` → 服务器 `git pull` → 重启后端 + 前端 build。
